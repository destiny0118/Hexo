## 安装hexo

```
npm install -g hexo-cli
```

## 初始化博客路径

```
hexo init blog
cd blog
npm install				# npm install命令会把X包安装到node_modules目录中

本地运行
hexo s -p 5555本地运行
```

##  远程部署

```
 npm install hexo-deployer-git --save	
```
